# Web3 Wallet Setup and DApp Interaction Task

## Objective

The objective of this task is to gain hands-on understanding of Web3 concepts by setting up a crypto wallet, configuring Ethereum testnet, deploying a smart contract, and interacting with decentralized applications (DApps).

## Implementation Summary

### 1. MetaMask Wallet Setup

* Installed MetaMask browser extension
* Created a new wallet account
* Stored seed phrase securely offline

### 2. Sepolia Testnet Configuration

* Enabled "Show Test Networks" in MetaMask
* Connected wallet to Sepolia Test Network

Sepolia ETH could not be obtained from available faucets due to restrictions.

### 3. Smart Contract Deployment

* Used Remix Ethereum IDE
* Developed and deployed **SimpleStorage** smart contract
* Execution environment: **Remix VM (Shanghai)**
* Performed:

  * `store(15)`
  * `retrieve()`

### 4. DApp Interaction

* Connected MetaMask wallet to Uniswap
* Explored:

  * Gas fee estimation
  * Wallet connection prompts
  * Transaction preview

(Token swap was not executed due to insufficient Sepolia ETH)
